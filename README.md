# Letterly 💌

Letterly is a creative web application designed for writing and sharing personalized digital letters. Users can craft heartfelt messages for their futureself and for their loved ones, customize them with aesthetic themes and stickers, and enjoy an interactive reading experience.

## Features

* 💌 Create personalized digital letters
* 🎨 Beautiful and responsive user interface
* 🖼️ Decorative sticker customization
* 📱 Mobile-friendly design
* ⚡ Interactive user experience using JavaScript
* 🌐 Deployed online with Netlify

## Tech Stack

* HTML5
* CSS3
* JavaScript
* Git & GitHub
* Netlify
